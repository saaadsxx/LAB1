Вопросы по 1 лб:
  1. Что такое HTML и для чего он используется?
    язык разметки гипертекста
    он нужен для создания разметки страницы в текстовом виде
    разметка - расположение элементов на странице
  
  2. Каковы основные компоненты HTML-документа?
    <!DOCTYPE html> - установка стандарта по которому написана страница
    <html></hmtl> сама страница
      <head></head> - метаданные о странице
        например title - название вкладки
        <meta charset="utf-8"> - кодировку текста
        font - шрифт
        <link rel="stylesheet" href="style.css"> - подключение скрипов
      <body> - сама разметка - здесь все элементы страницы

  3. В чем отличия между тегами, элементами и атрибутами?
    Теги - то, что образуют элемент. Это кусочки текста которые влияют на другой текст
    Элементы - то, что образовано элементами, то что отображено на экране, то что заполняет пространство страницы
    атрибут - это свойство элемента

  4. Как добавить комментарии в HTML-код?
    <! -- комментарий -->

  5. Как задать заголовок и подзаголовок в HTMLдокументе?
  h1-h6 - заголовки
  h1 - большой заголовок, остальные поменьше

  6. Как создать список в HTML?
    с помощью тегов <ul> <ol> <dl> (unordered, ordered, description list)
    пункты внутри списков <li> (list item)
      для <dl> <dt></dt><dd></dd> (data term data description)

  7. Как вставить изображение в HTML-документ?
    с помощью одиночного тега <img>
    и атрибута src="путь к файлу"
    alt="текст если изображение не загрузилось"

  8. Как создать ссылку на другой сайт или файл?
    использовать тег <a> с атрибутом href
    ссылка должна быть абсолютной для внешних фалов
    для внутренних можно просто полное имя файла
    <a href="ссылка">
    widht, height тоже прям внутри одинарного тега

  9. Как создать абзац?
    тег <p> (paraghraph)
    или писать текст а перенос сделать <br/>

  10.Как добавить таблицу в HTML-документ и задать ее размер?
      саму таблицу с помощью тега <table>
      а размер наверное количеством тегов <th>
      <td> уже автоматически вкладываются под свой <th>

  11.Как создать форму в HTML-документе и добавить в нее поля ввода данных?
    форма через <form>
    поле ввода данных <input type"text"> (ну или другой <input>)


  12.Как создать кнопку в HTML-документе?
    тег <button>text</button>
    основной атрибут - disabled

  13.Как создать гиперссылку на определенный раздел документа?
      использовать якоря - ссылки к элементу (через свойство id)
      пример создания ссылки ниже
      <h1 id="anchor">Якорь</h1>
      <a href="#anchor">Перейти к якорю</a>

  14.Как создать текстовую область в форме?
      для начала создаем форму тегом <form>
      если имеется ввиду область для ввода текста
      <input type="text">
      еще загуглил, можно <textarea> - тоже самое, но с возможностью переноса текста


  15.Как создать список с маркерами в HTML-документе?
    <ul> (unordered list)
      <li>
    </ul>

  16.Как создать список с нумерацией в HTML-документе?
    <ol> (ordered list)
      <li>
    </ol>

  17. Как создать кнопку отправки формы в HTML-документе?
    <input type="submit">
    по-идее это обычная кнопка без каких-либо эффектов

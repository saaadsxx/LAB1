Что такое позиционирование в CSS и какие значения свойства position вы знаете?
static — значение по умолчанию, элемент позиционируется на основе вложенности в родителей;
relative — элемент позиционируется относительно своего исходного положения в потоке документа;
absolute — элемент позиционируется относительно родительского элемента, который имеет позиционирование, отличное от static;
fixed — элемент позиционируется относительно окна браузера, но не может быть сдвинут;
sticky — элемент позиционируется относительно окна браузера, но может быть отодвинут от своей стандартной позиции.


Что такое flexbox
- это контейнер, который задает правила расположения вложенных в него элементов, который может менять размер в зависимости от их расположения
например установка вряд, колонкой, погоризонтали

Как работает свойство flex-grow?
относительно занимаемое пространство объекта

В чем разница между align-content, align-items, align-self
items - у контейнера на дочерние
self - на элемент на себя
content - по строкам контейнера на пачки элементом

Какие свойства имеют флекс-элементы?
· flex-grow:  — Определяет, насколько элемент может увеличиваться.
· flex-shrink:  — Определяет, насколько элемент может сжиматься.
· flex-basis:  — Определяет начальный размер элемента.
· order:  — Определяет порядок отображения элемента в контейнере.
· align-self:  —  Переопределяет выравнивание элемента по перекрестной оси.

Что такое адаптивная верстка?
Адаптивная вёрстка сайта — это способ обеспечить отображение интерфейсов на устройствах с разным разрешением экранов.


Что такое резиновая верстка?
Резиновая вёрстка (liquid, fluid) — вид вёрстки, при которой содержимое растягивается в зависимости от размера окна. Размеры задаются не в пикселях, а в процентах от величины экрана, и затем пропорционально масштабируются.

Что такое медиазапросы? Для чего применяется?
это набор правил, которые позволяют адаптировать внешний вид веб-страницы под технические параметры устройства пользователя

Какие директивы используется для медиазапросов?
@media - в нее можно передавать функции и операторы

Какие значения можно передать в директиву @media?
параметры уникальные для устройств устройств
соотношение сторон экрана
например минимальное и максимальное разрешение
ориентация экрана
способ прокрута экрана

В каком порядке надо располагать директивы @media?
в порядке убывания по max-параметру
в порядке увеличения по min-параметру

Что такое бургер меню?
Бургер-меню – это способ представления навигационного меню на устройствах с маленьким экраном.
При нажатии на иконку бургер-меню разворачивается, показывая список пунктов меню, и при повторном нажатии сворачивается.

Что такое навигационное меню?
Навигационное меню – это область интерфейса с навигационными опциями, позволяющие пользователям находить контент на сайте.
